# Html.md

## Core Rules

- Never delete or overwrite existing code unless explicitly asked
- Never change visual design, colors, fonts or layout
- Never switch from CDN to npm or any build tool
- Never create subfolders — all files stay in root
- Never use absolute paths, only relative
- Never duplicate header/footer HTML — use shared.js
- Always keep shared.js as the last tag before </body>
- Always use relative links between pages: href="chapter2.html"
